# 概要
旅用携帯として購入。旅しててもスマホばっか見ていて、旅に集中しきれていないことに疑問を感じてたんですよね。
コンパクトで軽量なのもいいですが、ボタン一つひとつがとても押しやすいのが一番お気に入りです。電池持ちもよくて、2泊3日なら充電器不要でした。
これを使って旅をしてみると、それまでなにがあってそんなにスマホ見てたんだろうということに気づけていいです。
BluetoothだけでなくWi-Fiにも非対応なぐらい割り切って作られていますが、実際に使ってみるとそれも納得できるものだと感じます。

# アプリのインストール方法
「開発者向けオプション」を有効にしてADBでAPKを入れることができますが、手順が特殊です。

まずPC内にあるAPKファイルをこの端末の「ダウンロード」に送ります。ファイル名に記号があると次の手順で詰まるのでなるべくシンプルなファイル名にすることをおすすめします。
ファイルの転送が終わったらshellに入れておきます。
```shell
adb push (PC内のAPKファイルの絶対パス) /sdcard/Download/
adb shell
```

```shell
pm install-create -i ringo
```

すると以下の文字列が表示されます。[]内の数字を次の手順でコピペしてください。なお、一文字でも打ち間違えた場合はもう一度最初からやり直しです。

```shell
Success: created install session [0000000000]
```

```shell
pm install-write 0000000000 gorira /sdcard/Download/(ファイル名).apk
```

```shell
pm install-commit 0000000000
```

あとは「ツール」→「アプリ一覧」でアプリが追加されていれば成功です。
